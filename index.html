Command line tools

Useful for Bash scripting

shelltools - simple utilities for command line and shell scripting

  • findfile - find files based on prefix, suffix on name contents
  • finddir - find directories based on prefix, suffix or name content
  • mergepath - merge a path, without duplicates into a PATH list
  • range - emit a range of integers
  • reldate - calculate a relative date in YYYY-MM-DD format
  • timefmt - format time output per Go’s time package time notation
  • urlparse - extract URL parts (e.g. protocol, hostname, path)

datatools - a small set of tools to facilitate working with CSV, JSON and Excel Workbook data in Bash scripts

  • csvcols - filter or render CSV data rows
  • csvjoin - join two CSV tables on a designated column value
  • csv2mdtable - render a CSV table as a Github Flavored Markdown table
  • jsoncols - using dot path notation extract JSON object elements into columns
  • jsonrange - iterate of a JSON object or array
  • xlsx2json - convert an Excel Workbook sheet into a JSON document
  • xlsx2csv - convert an Excel Workbook sheet into into a JSON document

Content formatting tools

  • mkpage - a deconstructed content system for static content publishing
    • mkpage - a light weight markdown and template processor for multiple data sources and types
    • mkslides - a markdown to HTML page presentation renderer
    • mkrss - a RSS generator for Markdown content
    • sitemapper - a sitemap.xml generator
    • reldocpath - calculate relative file paths
    • titleline - extract the first title from a Markdown file
    • byline - extract a byline from an article written in Markdown
    • slugify - transform a title into a path/url friendly string
    • urlencode and urldecode - url encode/decode from the command line
    • ws - A nimble static content webserver
      • doesn’t serve dot files (e.g. .htaccess) or dot directories (e.g. .git)
      • fast startup
      • logs to console so you can watch requests and responses
  • opml - A naive OPML parse package that features a OPML sort utility
  • prettyxml - a naive XML pretty printer
  • shorthand - a text label expander with markdown processor support
  • stngo - Standard Timesheet Notation processor for my personal project time logs
