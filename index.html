Command line tools

Useful for Bash scripting

shelltools - simple utilities for command line and shell scripting

  • csvcols - turn command line parameters into a CSV formatted row
  • findfile - find files by prefix, suffix or containing a character string
  • finddir - find directories by prefix, suffix or containing a characer string
  • jsoncols - pull values from a json blob and list them as columns
  • mergepath - prefix, append, clip path lists (e.g. $PATH)
  • pathparts - split a path into parts (e.g. directory, basename, extension)
  • reldate - display relative dates in YYYY-MM-DD format
  • range - renders a list of numbers in a range separated by spaces (useful to use in for loops in Bash)
  • timefmt - wraps the Golang time formatting package so you can use those expressions in your Bash script
  • urlparse - utlity to make it easy to parse and work with URLs in Bash (or your favorite shell)

Content formatting tools

  • mkpage - a light weight go template and markdown processor
  • mkslides - turns a single markdown file into a series of HTML webpages for presentations
  • opml - A naive OPML parse package that features a OPML sort utility
  • prettyxml - a naive XML pretty printer
  • shorthand - a text label expander with markdown processor support
  • stngo - Standard Timesheet Notation processor for my personal project time logs

web services

  • ws - A simple webserver for development use (e.g. you can serve your current work directory out on localhost:8000)
    • doesn’t serve dot files (e.g. .htaccess) or dot directories (e.g. .git)
    • fast startup
    • logs to console so you can watch requests and responses
