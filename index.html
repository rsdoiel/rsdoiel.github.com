Command line tools

Useful for Bash scripting

  • findfile - two command line utilities findfile and finddir offer simplier options for the standard Unix find
  • mergepath - output a path based on new path elements without injecting repeats
  • pathparts - Exposes path elements in Bash based on the Golang path package
  • range - renders a list of numbers in a range separated by spaces (useful to use in for loops in Bash)
  • reldate - renders a relative date in YYYY-MM-DD format
  • timefmt - wraps the Golang time formatting package so you can use those expressions in your Bash script
  • urlparse - utlity to make it easy to parse and work with URLs in Bash (or your favorite shell)

General purpose

  • md2slides - turns a single markdown file into a series of HTML webpages for presentations
  • opml - A nieve OPML parse package that features a OPML sort utility
  • prettyxml - a XML pretty printer
  • shorthand - a text label expander with markdown processor support
  • stngo - Standard Timesheet Notation processor for my personal project time logs
  • ws - A simple webserver for development use (e.g. serves your current work directory out as localhost:8000)
